PlotToolsAssist

PlotToolsAssist is a helper tool for the game Palia designed to help players manage and optimize the use of space in their Home Plot. It provides statistical insights to track, organize, and analyze items across your inventory, storage, lockbox, and plot.

Features
-Identify items in Inventory / Storage / Lockbox but not on the Plot
-Identify items on the Plot but missing from Inventory
-Detect duplicate items
-Track missing collectibles
-Generate statistical summaries to optimize plot space efficiently

Dependencies
Python 3.11+
Selenium
Palia Assistant (by PaliaPedia developers)
⚠️ These tools are only required to fetch and process item data. PlotToolsAssist itself is purely for statistical analysis.
