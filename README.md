# PlotToolsAssist

![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)
![Python](https://img.shields.io/badge/Python-3.11+-blue.svg)
![Selenium](https://img.shields.io/badge/Selenium-Required-orange.svg)

**PlotToolsAssist** is a helper tool for the game **Palia**, designed to help players **manage and optimize their Home Plot** efficiently. It provides statistical insights to track, organize, and analyze items across your inventory, storage, lockbox, and plot.

---

## Features

- ✅ Track items across **Inventory, Storage, Lockbox, and Plot**  
- ✅ Generate lists of **items not yet placed on the Plot**  
- ✅ Detect **duplicate items**  
- ✅ Identify **missing collectibles**  
- ✅ Generate **statistical summaries** to optimize plot space efficiently

- **Python 3.11+ (embedded)**   
- **Selenium (embedded)**   
- **Palia Assistant (embedded)**  

> ⚠️ These tools are only required to fetch and process item data. PlotToolsAssist itself is purely for statistical analysis.
